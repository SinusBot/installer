Installation

Alternatively download the install here script manually.

Information

For Debian 8+, CentOS 7+, Ubuntu 16.10+

The installer does the following:

  • install latest beta bot
  • install latest TS3 client
  • install all necessary aptitudes
  • create seperate sinusbot user
  • install YT-Dl (optional)
  • cronjob for daily YT-Dl update and Sinusbot (once supported)
  • change file permissions
  • generate systemd file => start/stop with "service sinusbot {start|etc}"
  • delete unneeded files and archives
  • starts sinusbot after installation
  • option to update and remove the bot
